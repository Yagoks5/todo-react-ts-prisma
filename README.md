## Instale as dependencias:
# Na pasta backend
```bash
cd backend
npm install
```


# Na pasta frontend
```bash
cd frontend
npm install
```



Execute as migrações do prisma:
# Na pasta backend
```bash
npx prisma migrate dev
```



## Inicie o servidor
# Na pasta backend
```bash
npx ts-node server.ts
```


## Inicie a aplicação:
# Na pasta frontend
```bash
npm run dev
```
